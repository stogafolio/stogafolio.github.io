
# The Folio Website

The website for the Conestoga folio

## Instructions

You'll want to first clone this repository. You can do this with the command `git clone https://github.com/stogafolio/website`. If you're using VSCode, you can use the [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) extension to view the site.
